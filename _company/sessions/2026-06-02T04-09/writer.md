CEO님의 지시를 바탕으로 Mini-Blueprint의 핵심 가치인 '통제감'을 극대화하고, 구매 결정에 필요한 모든 심리적 장치를 갖춘 최종 SBA 카피라이팅 묶음을 완성했습니다. 이 카피는 단순히 버튼 문구를 넘어, 사용자가 스스로 결제를 '필요한 행동(Necessary Action)'으로 인식하도록 설계된 구조입니다.

### 🚨 Mini-Blueprint: Single Best Action (SBA) 카피라이팅 패키지
**(목표: 미조치 시의 손실 공포(FOMO) 자극 $\rightarrow$ Mini-Preview 구매 유도)**

#### 📝 [공통 흐름 구조] - 모든 CTA에 적용될 스토리텔링 순서
1. **Pain Acknowledgment (불안감 인정):** "막연히 불안한가요? 아닙니다. 데이터가 알려주는 명확한 위험이 있습니다."
2. **The Proof (데이터 제시 및 공포 자극):** (Mockup 기반) "당신의 현재 리스크 지표는 [X점]입니다. 이 점수는 지난 6개월간 평균 대비 [Y%] 높은 수치이며, 이것은 곧 \[구체적인 손실 예시: 기회 비용, 시간, 자산 등]이 누적되고 있다는 것을 의미합니다."
3. **The Gap (해결책의 부재):** "문제는 이 지표가 '상관관계'만을 보여준다는 것입니다. 막연한 경고에만 머무른다면, 다음 분기에도 같은 위험을 반복할 수밖에 없습니다."
4. **Mini-Blueprint 제시 (유일한 해결책 포지셔닝):** "이 리스크를 0으로 만들 단 하나의 '최소 개선 경로'와 즉시 실행 가능한 행동 계획(SBA)이 필요합니다. 그것이 Mini-Blueprint가 제공하는 핵심입니다."

---

#### 🚀 [CTA 버전별 상세 카피 및 연출 지침]
각 CTA는 다른 심리적 레버리지(공포, 논리, 긴급성)를 건드립니다. 웹페이지의 섹션 위치에 따라 적절한 버전을 선택하여 사용하십시오.

##### **📌 Version A: 공포/FOMO 기반 (가장 강력함 - '위험'이 느껴지는 지점 배치)**
*   **타겟 감정:** 패닉, 손실 회피 본능 (Loss Aversion)
*   **카피 톤:** 경고적, 단호함. 질문형으로 마무리하여 행동을 강제함.

| 구성 요소 | 카피라이팅 제안 | 연출 지침 (UX/UI 가이드) |
| :--- | :--- | :--- |
| **Pre-CTA 헤드라인** | "이 리스크, 방치하면 1년 뒤 당신의 포트폴리오는 [X%] 낮아집니다." | 배경에 깜빡이는 경고색(Red/Amber) 사용. 자막 효과 필수. |
| **본문 유도 문구** | "현재 데이터만으로는 '어디서부터' 손을 대야 할지 모릅니다. 막연한 불안함은 공포일 뿐, 해결책이 아닙니다. 지금 당장 취해야 할 단 하나의 행동 지침(SBA)이 필요합니다." | 텍스트 박스에 '❌ 막연한 걱정'과 '✅ 구체적인 액션 플랜'을 대비시켜 보여줍니다. |
| **CTA 버튼 문구 (Primary)** | **[🚨 지금, 최대 리스크를 통제하세요]** | 배경색: 강렬한 레드/오렌지. 마우스 오버 시 진동 효과 부여. |
| **보조 CTA 텍스트** | "Mini-Preview로 정확한 개선 경로(SBA)를 확인하고, 손실을 막는 첫 단추부터 채우세요." | 버튼 아래 작은 글씨로 배치하여 신뢰도를 높입니다. |

##### **📌 Version B: 논리/통제 기반 (신뢰도 확보가 필요한 지점 배치)**
*   **타겟 감정:** 합리적 탐색, 통제감 획득 욕구 (Control Seeking)
*   **카피 톤:** 전문적, 분석적. '지표'와 '근거'를 강조하여 신뢰도를 높입니다.

| 구성 요소 | 카피라이팅 제안 | 연출 지침 (UX/UI 가이드) |
| :--- | :--- | :--- |
| **Pre-CTA 헤드라인** | "막연한 예측은 이제 그만. 데이터 기반의 '최소 개선 경로'를 발견하세요." | 전문적인 차트나 그래프 이미지와 함께 배치. (지적 만족감 자극) |
| **본문 유도 문구** | "우리는 상관관계를 보여줄 뿐, 인과관계까지 단정할 수 없습니다. 하지만 Mini-Blueprint는 다릅니다. 핵심 근거(Source/Basis)를 기반으로 '가장 효율적인 다음 스텝'을 구조화하여 제공합니다." | 리스크 지표 계산 과정의 복잡함이 사용자에게 이득이 되는 과정을 간결하게 설명해야 합니다. |
| **CTA 버튼 문구 (Primary)** | **[📊 내 자산의 가장 약한 고리를 분석하기]** | 배경색: 신뢰감을 주는 딥 블루/네이비. 데이터 그래프와 어울리게 디자인합니다. |
| **보조 CTA 텍스트** | "단일 행동 지침(SBA)을 통해 리스크를 구조적으로 해소하는 방법을 확인하세요." | 버튼 아래에 '데이터 분석 엔진 기반' 같은 전문 용어를 넣어 권위를 높입니다. |

##### **📌 Version C: 긴급성/기회비용 기반 (구매 임계점 직전 배치)**
*   **타겟 감정:** FOMO, 기회의 상실 공포 (Opportunity Loss)
*   **카피 톤:** 시간 제한적, 독점적. 즉시 행동하지 않으면 남이 이득을 본다는 느낌 부여.

| 구성 요소 | 카피라이팅 제안 | 연출 지침 (UX/UI 가이드) |
| :--- | :--- | :--- |
| **Pre-CTA 헤드라인** | "남들이 이미 시작한 '최소 개선 경로'를 놓치지 마세요. 타이밍이 가장 중요합니다." | 시계나 카운트다운 타이머 이미지를 활용하여 긴급성을 부여합니다. |
| **본문 유도 문구** | "경쟁자들은 현재의 리스크 점수를 바탕으로 '가장 빠르게 수익을 확보할 수 있는 단일 행동'을 실행하고 있습니다. 당신이 망설이는 그 순간, 시간과 자원은 흘러갑니다." | 이 섹션에서는 경쟁사 또는 동료들의 성공 사례(익명화된)를 간접적으로 언급하는 것이 효과적입니다. |
| **CTA 버튼 문구 (Primary)** | **[⏱️ 남들보다 먼저 통제권을 확보하기]** | 배경색: 금속성 골드/브론즈 톤을 사용하여 가치와 희소성을 부여합니다. |
| **보조 CTA 텍스트** | "선착순 Mini-Preview 혜택을 놓치지 마세요. 지금 바로 당신의 Action Plan을 확보하세요." | 시간 제한(예: '오늘 자정까지') 문구를 반드시 명시해야 합니다. |
