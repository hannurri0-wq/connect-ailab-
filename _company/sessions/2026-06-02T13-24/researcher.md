모든 오류 메시지는 **"문제가 발생했습니다."**가 아니라, **"현재 상황을 분석하고 개선할 수 있는 기회입니다."**라는 뉘앙스로 전달되어야 합니다.

### 1. 결제 및 거래 실패 시나리오 (Payment Flow Failures)

| 기술 오류 유형 | 사용자가 겪는 상황/원인 | 표시할 사용자 메시지 (톤&매너) | 핵심 행동 유도/다음 단계 |
| :--- | :--- | :--- | :--- |
| **카드 거절 / 한도 초과** | 결제한 카드의 잔액 부족, 카드사 승인 실패 등. (가장 흔함) | "결제가 완료되지 않았습니다. 죄송합니다." - 원인 분석: 사용하신 카드사 또는 계좌에 일시적인 문제가 있을 수 있습니다. | 1. **[필수]** 다른 결제 수단(카드/계좌이체)을 이용해 주세요.<br>2. **[대안]** 시간이 지난 후 다시 시도하거나, 고객센터를 통해 정확한 거절 사유를 확인해주세요. |
| **결제 게이트웨이 시간 초과 (Timeout)** | 네트워크 지연 또는 결제 시스템 간의 통신 오류. | "시스템 연결에 문제가 생겼습니다." - 원인 분석: 저희 서버와 결제사 간의 연결 상태가 일시적으로 불안정할 수 있습니다. 잠시만 기다려 주십시오. | 1. **[재시도]** '결제하기' 버튼을 다시 눌러주세요. (자동 재시도 로직 권장)<br>2. **[대안]** 만약 두 번 이상 실패한다면, 브라우저를 새로고침(F5)한 후 처음부터 다시 시도해 주세요. |
| **세션 만료 (Session Expired)** | 사용자가 장시간 동안 결제 페이지에 머무르는 경우 등. | "보안을 위해 접속 상태가 종료되었습니다." - 원인 분석: 고객님의 정보 보호를 위해 현재 세션이 자동 종료되었습니다. 불편을 드려 죄송합니다. | **[필수]** 처음부터 다시 진단 리포트를 요청해 주세요. (데이터 무결성 확보) |

### 2. 서비스 API 및 데이터 처리 오류 시나리오 (Data & Logic Failures)

| 기술 오류 유형 | 사용자가 겪는 상황/원인 | 표시할 사용자 메시지 (톤&매너) | 핵심 행동 유도/다음 단계 |
| :--- | :--- | :--- | :--- |
| **Z Score API 호출 실패 / 네트워크 단절** | 서버 연결 불가, 외부 데이터 공급처 문제 등. | "현재 리스크 점수 측정에 오류가 발생했습니다." - 원인 분석: 저희 시스템이 최신 데이터를 가져오는 데 일시적인 문제가 있습니다. 전문 진단 과정 중 잠시만 기다려 주십시오. | 1. **[재시도]** '진단 재실행' 버튼을 눌러주세요.<br>2. **[대안]** 오류가 지속될 경우, 인터넷 연결 상태를 확인해 주세요. |
| **필수 입력값 누락 (Validation Error)** | 사용자가 필수 정보를 기입하지 않고 다음 단계로 넘어간 경우. | "진단 분석을 위해 몇 가지 정보가 필요합니다." - 원인 분석: 더 정확한 리스크 측정을 위해서는 [누락된 항목]이 반드시 필요합니다. | **[강조]** 굵게 표시된 빈칸에 빠짐없이 정보를 기입해 주십시오. (구체적인 가이드 제시) |
| **데이터 비정형/미지원 오류** | 시스템이 예상치 못한 포맷의 데이터를 받거나 처리하지 못하는 경우. | "분석 가능한 데이터 범위를 벗어났습니다." - 원인 분석: 현재 입력하신 [유형] 데이터는 저희 진단 모델에서 지원하지 않는 방식입니다. | **[가이드 요청]** 해당 정보를 텍스트로 풀어서 설명해주시거나, 다른 형태의 자료를 첨부해 주시면 재분석을 도와드리겠습니다. |

### 3. 기타 시스템 오류 및 공통 처리 (General System Failures)

| 기술 오류 유형 | 사용자가 겪는 상황/원인 | 표시할 사용자 메시지 (톤&매너) | 핵심 행동 유도/다음 단계 |
| :--- | :--- | :--- | :--- |
| **서버 내부 오류 / 시스템 점검** | 예상치 못한 서버 다운 또는 정기 점검. | "서비스 이용에 불편을 드려 죄송합니다." - 원인 분석: 현재 저희가 시스템 안정화 작업을 진행하고 있습니다. 최대한 빠른 시간 내에 정상 서비스를 재개하겠습니다. | 1. **[알림]** 잠시 후 다시 시도해 주십시오.<br>2. **[문의]** 긴급 문의는 고객센터를 이용해 주시면 감사하겠습니다. |
| **권한 부족 (Access Denied)** | 사용자가 접근 권한이 없는 페이지에 접속하거나, 유료 서비스 미구독 상태인 경우. | "접근할 수 없는 영역입니다." - 원인 분석: 이 기능은 [Premium/유료] 회원에게만 제공되는 전문 진단 과정입니다. | **[명확한 경로 제시]** 'Mini-Blueprint Pro' 구독을 통해 기능을 활성화하거나, 전문가와 상담하는 옵션을 이용해 주세요. |
